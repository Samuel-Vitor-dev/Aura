# Aura

Uma landing page moderna e envolvente para estúdios criativos, portfolios e produtos digitais — pronta para mostrar seu trabalho, captar leads e converter visitantes em clientes.

**Destaques**
- Design responsivo e visual premium
- Animações leves em JavaScript para maior impacto
- Estrutura simples e fácil de personalizar
- Arquivos e assets organizados em `src/` para desenvolvimento

**Por que usar este template**
- Converte: layout pensado para mostrar portfólio e chamadas para ação claras
- Rápido de configurar: abra `index.html` ou sirva com um servidor estático
- Fácil de adaptar: CSS e JS separados para customização rápida

**Demo local (rápido)**
1. Abrir localmente:

```bash
# Basta abrir o arquivo no navegador
start index.html
```

2. Ou servir via um servidor estático (recomendado para testes):

```bash
npm install -g serve
serve -s .
```

**Pronto para deploy**
- GitHub Pages: faça push para um repositório e ative Pages nas configurações
- Netlify / Vercel: arraste e solte a pasta do projeto ou conecte o repo para deploy contínuo

**Arquivos importantes**
- Página principal: [index.html](index.html)
- Estilos: [src/css/styles.css](src/css/styles.css)
- JavaScript principal: [src/js/main.js](src/js/main.js)
- Outras folhas e scripts: veja `src/assets/` para imagens e recursos

**Customização rápida**
- Troque cores e tipografia em [src/css/styles.css](src/css/styles.css)
- Edite textos e seções em `index.html`
- Ajuste interações em [src/js/main.js](src/js/main.js)

**Pitch para vendas**
Use este site como uma vitrine profissional: destaque projetos com imagens de alta qualidade, adicione depoimentos e uma chamada para ação clara (contato/contratação). Ideal para freelancers, estúdios e lançamentos de produto.
